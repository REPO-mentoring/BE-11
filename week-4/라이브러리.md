# 라이브러리

## 1. java.lang 패키지

자바 프로그램의 기본적인 클래스들을 담고 있으며, 별도의 import 없이 사용할 수 있습니다.

### 1.1 Object 클래스

모든 자바 클래스의 최상위 부모 클래스입니다.

- equals : 객체의 번지를 비교합니다. 논리적 동등성을 위해 주로 Overriding합니다.
- hashCode(): 객체의 해시코드를 리턴합니다. HashSet, HashMap 등에서 객체 식별자로 쓰입니다.
- toString() : 객체의 문자열 정보를 리턴합니다. (보통 클래스명@16진수해시코드 형태)

### 1.2 System 클래스

운영체제의 기능을 일부 이용할 수 있게 해줍니다.

- exit() : 프로그램 강제 종료.
- currentTimeMillis(), nanoTime(): 실행 시간 측정 시 유용.
- getProperty() : 시스템 속성 읽기.

### 1.3 String & StringBuilder

- **String:** 문자열을 수정할 수 없는 특성이 있어, 새로운 문자열을 더할 때마다 새로운 객체가 생성됩니다.
- **StringBuilder:** 내부 버퍼에 문자열을 저장하며 수정 가능합니다. **문자열 결합 연산이 많을 때** 성능 면에서 유리합니다.

## 2. Wrapper 클래스

기본 타입을 객체로 다루기 위해 포장하는 클래스입니다.

- **Boxing:** 기본 타입 → 포장 클래스
- **Unboxing:** 포장 클래스 → 기본 타입
- Java 5부터는 **Auto Boxing/Unboxing**이 지원되어 자동으로 변환됩니다. (예: Integer obj = 10;)

---

## 3. Math & Class 클래스

- **Math:** 수학 계산을 위한 정적 메소드 제공 (abs, ceil, floor, max, random 등).
- **Class:** 클래스와 인터페이스의 메타 데이터(이름, 필드, 메소드 정보)를 관리합니다. 리플렉션의 기초

---

## 4. java.util 패키지

프로그램 개발에 보조적인 역할을 하는 클래스들이 모여 있습니다.

### ① Arrays 클래스

배열 조작(복사, 항목 정렬, 항목 검색)을 위한 메소드를 제공합니다.

- Arrays.sort() : 배열 정렬.
- Arrays.binarySearch(): 이진 탐색.

### ② Random 클래스

다양한 타입의 난수를 생성할 때 사용합니다. Math.random()보다 세밀한 제어가 가능합니다.

### ③ Date & Calendar 클래스

날짜와 시간 정보를 얻을 때 사용합니다. 

---

## 5. 정규 표현식 (Pattern 클래스)

문자열이 정해진 패턴(전화번호, 이메일 형식 등)과 일치하는지 확인할 때 사용합니다. Pattern.matches() 매소드 이용.
